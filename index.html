Compras
Implementado v1

Modelo que cruza la demanda real (promedios móviles 3M/6M/12M) con el stock físico para decidir qué, cuánto y cuándo comprar. Menos quiebres y menos capital inmovilizado.

Costos de Importación

Compras / Comex
Implementado v1

Costo total importado por pedido con comparación contra el promedio histórico de 12 meses, para detectar desviaciones por producto.

Pedidos ETA

Ventas / Comex
Implementado v1

Seguimiento de importaciones, vencimientos y glosas de pago, centralizando lo que antes vivía en múltiples planillas.

Validación SII vs ERP Manager

Finanzas / Tributario
Primera versión

Control tributario que cruza el libro de compras del SII con el ERP, detectando diferencias de total e IVA y facturas faltantes.

Importación XML → ERP

Finanzas / Tributario
Prototipo

Lee el XML de la factura electrónica del SII y genera el archivo de importación al ERP con reglas por proveedor, reduciendo el ingreso manual.

Conexión ERP → Azure SQL

Soporte TI / Infraestructura
Diseño en curso

Automatización de la conexión directa del ERP Manager a Azure SQL con refresco diario, para alimentar los paneles BI sin pasos manuales.
